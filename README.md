# Internal Business Automation – Iron Lady

## Overview
This project is an internal Learner Management System designed for Iron Lady’s operations team. It helps manage learner records efficiently using a web-based CRUD application.

## Problem
Manual tracking of learners using spreadsheets and messages is time-consuming and error-prone.

## Solution
A centralized system that allows admins to create, view, update, and delete learner records.

## Features
- Add learner
- View learner list
- Update learner status
- Delete learner record

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- SQLite database
